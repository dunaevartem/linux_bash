DevOps Workspace Toggler
Минималистичный Bash-скрипт для автоматического развертывания рабочего окружения в tmux. Одной командой устанавливает нужный софт, конфигурирует терминал и запускает панель мониторинга.
Что делает скрипт?
Проверка зависимостей: Автоматически устанавливает tmux, htop и mc, если они отсутствуют в системе.
Настройка Tmux: Создает файл ~.tmux.conf с поддержкой мыши и удобной навигацией (Vim-style).
Логика Toggle:
Если сессия devops уже запущена — завершает её.
Если сессии нет — создает новую с разделением на 4 окна:
mc (Midnight Commander)
htop (Мониторинг ресурсов)
df -h (Мониторинг дисков через watch)
Свободный терминал
Использование
Дай скрипту права на выполнение и запусти его:
```
bash
chmod +x setup_workspace.sh
./setup_workspace.sh
```
Горячие клавиши (Tmux)
Благодаря конфигу в скрипте, навигация по панелям работает через Ctrl+b +:
h — влево
j — вниз
k — вверх
l — вправо
Мышь — включена (можно менять размер панелей и выбирать их кликом).
Скрипт предназначен для систем на базе Debian/Ubuntu (использует apt).
